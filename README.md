# 📄 Уведомления по форме №591/пр (ИЖС)

Три готовые HTML-формы для заполнения и печати уведомлений в Департамент архитектуры.

## Какие формы входят в комплект

| Файл | Приложение | Назначение |
|------|------------|------------|
| `pril1_plan.html` | Приложение №1 | Уведомление о планируемом строительстве или реконструкции |
| `pril4_izmenenie.html` | Приложение №4 | Уведомление об изменении параметров строительства |
| `pril5_okonchanie.html` | Приложение №5 | Уведомление об окончании строительства |

## Возможности

- ✏️ Режим редактирования прямо в браузере
- 📸 Загрузка схемы участка (клик по рамке)
- 📄 Экспорт в PDF с сохранением всех данных
- 🔄 Автоматическая синхронизация ФИО и типа объекта
- 🖨️ Печать без красных рамок и служебных кнопок

## Как пользоваться

1. Откройте `index.html`, выберите нужную форму
2. Заполните красные поля
3. При необходимости загрузите схему участка
4. Нажмите «Скачать PDF» или «Печать»

## Технические детали

- Всё работает локально, без интернета (кроме библиотеки `html2pdf.js` с CDN — можно заменить локальной копией)
- Данные никуда не отправляются
- PDF генерируется в браузере

## Адаптация под свой регион

- Департамент архитектуры — редактируется
- Адрес и телефон — редактируются
- Все примеры данных (кадастровый номер, отступы и т.д.) — примерные, замените на свои
